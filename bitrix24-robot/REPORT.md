# report
Что я узнала о Локальных приложениях в Битрикс24 и Роботах в Битрикс24

Локальные приложения в Битрикс24
Есть несколько типов локальных приложений:
Серверное - имеет собственный интерфейс, размещается на внешнем сервере, работает через OAuth 2.0 авторизацию
Приложения только с использованием API - не имеет визуального интерфейса, выполняет задачи через  REST API, работает в фоновом режиме.
Статическое приложение на остнове HTML/JS - не требует внешнего сервера, работает на фронтенде.

Роботы в Битрикс24
Также есть несколько видов:
Роботы для работы с CRM(система управления взаимоотношениями с клиентами)- создание,удаление, добавление, изменение сделок,контактов
Роботы для коммуникаций - отправление смс, отправка писем на почты, уведомления,отправление сообщений в открытую линию, отправлять сообщения в мессенджеры
Роботы для управления задач - создаёт, изменяет, завершает, получает инфо о задаче
Роботы для документов - создаёт документ по шаблону CRM, генерирует документы; создаёт документ и отправляет на подпись.
Роботы для бизнес-процессов - запускает бизнес-процесс, запускает бизнес-процесс
Служебные - изменяет ответственного, изменяет наблюдателя, ожидает оплату, генерирует ссылку для оплаты, вычисляет, отправляет данные из вебхука (URL)
Рекламные роботы - добавляет рекламы в определённое место( ВК, Гугл)
Роботы для работы с AI - заполняет поля, обрабатывает текста, расшифровка звонков
